#### 1.lambda函数(定义简单函数)

语法:	lambda 参数名称: 简单表达式 , 	

案例: add = lambda x,y: x + y

#### 2.filter函数(筛选可迭代对象元素)

语法 : filter(函数, 可迭代对象)

案例:  result = filter(lambda x: x % 2 == 0, nums)

注意:此时result 是一个filter对象,需要转化为list才能看到具体元素

#### 3.map函数(函数映射到迭代对象每个元素)

语法:  map(函数, 可迭代对象)`

案例: result = map(lambda x: x * 2, nums) 

#### 4.max,sort方法自定义

max(迭代对象,key = lambda)实现按自定义找最大(字典默认遍历key)

sort(list,key = lambda)实现自定义排序

#### 5.**`if __name__ == '__main__'`**：

这行代码的作用是 “只有直接运行这个文件时，才执行里面的代码”，如果这个文件被其他文件导入，测试代码不会执行，是 Python 里的最佳实践

nums = [10,20,30]
